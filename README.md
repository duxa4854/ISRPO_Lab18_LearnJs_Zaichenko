## Краткое описание работы

### Описание (что изучили)

В данной лабораторной работе были изучены основы программирования на JavaScript:

- **Переменные и константы** - объявление с помощью `let`, `const`, `var`
- **Типы данных** - динамическая типизация, примитивные и ссылочные типы
- **Операторы** - арифметические, логические, операторы сравнения
- **Условные конструкции** - `if/else`, `switch/case`, тернарный оператор
- **Циклы** - `for`, `while`, `do-while`, `for...of`, `for...in`
- **Функции** - объявление, стрелочные функции, параметры
- **Работа с консолью** - вывод через `console.log()`

## Итоговая таблица: JavaScript vs C#

| Аспект              | C#                           | JavaScript                   |
| ------------------- | ---------------------------- | ---------------------------- |
| **Типизация**       | Статическая                  | Динамическая                 |
| **Компиляция**      | Да (в IL-код)                | Нет (интерпретация/JIT)      |
| **Точка входа**     | Main()                       | Отсутствует                  |
| **Переменные**      | `int x = 5;`                 | `let x = 5;`                 |
| **Константы**       | `const int X = 5;`           | `const X = 5;`               |
| **Вывод в консоль** | `Console.WriteLine()`        | `console.log()`              |
| **Проверка типа**   | Во время компиляции          | `typeof`                     |
| **Сравнение**       | `==` (по значению)           | `===` (строгое)              |
| **Условия**         | `if/else/switch`             | `if/else/switch + тернарный` |
| **Циклы**           | `for/while/do-while/foreach` | `for/while/do-while/for..of` |

## Основные выводы

1. **JavaScript** - интерпретируемый язык с динамической типизацией
2. Не требуется явное указание точки входа (код выполняется сверху вниз)
3. Для объявления переменных используются `let` и `const`
4. Строгое сравнение `===` проверяет и значение, и тип
5. Тернарный оператор удобен для коротких условий
6. JavaScript выполняется в браузере или через Node.js
